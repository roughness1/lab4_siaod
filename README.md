# Подсчет частоты слов в тексте

Скрипт для анализа текстовой строки и подсчета количества упоминаний каждого слова. Решение реализовано на языке Python.

## Функционал

* **Игнорирование регистра:** Автоматическое приведение текста к нижнему регистру (слова `Cat`, `caT` и `cat` учитываются как одно и то же слово).
* **Очистка от пунктуации:** Удаление знаков препинания (`.`, `,`, `!`, `?`, `;`, `:`, `-`) по краям слов.
* **Эффективный подсчет:** Использование встроенного метода словаря `.get()` для безопасного инкремента счетчика.

## Пример работы

**Входная строка:** `"cat, dog, caT!??, bIrd, Dog!, cAt."`

**Вывод программы:** 
```text
cat -> 3
dog -> 2
bird -> 1
```

## Алгоритмическая сложность

* **Временная сложность:** $O(N)$, где $N$ - количество слов в тексте. Каждый шаг (приведение к регистру, разбиение, очистка и подсчет) выполняется за линейное время.
* **Пространственная сложность:** $O(U)$, где $U$ - количество уникальных слов, сохраненных в итоговом словаре.
